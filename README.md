# RegexAuthenticator

RegexAuthenticator is a Python script that allows users to input a regular expression and then test strings against that regular expression.

## Usage

1. Clone the repository to your local machine.

2. Navigate to the directory containing the `regex_authenticator.py` file.

3. Run the script using Python:

4. Follow the prompts:
- Enter a regular expression when prompted. For example: `\d{3}-\d{3}-\d{4}`.
- Enter a string to test against the regular expression. Press Enter to exit the program.
